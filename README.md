# youtube-clone-coding
make youtube web page using React

</br>
</br>


🚩 HTML, CSS, JS에 익숙해지고, React를 처음 사용해보며 학습하기 위해 진행했던 '유튜브 클론코딩'프로젝트입니다.


</br>
</br>


### 메인 페이지

<img src="https://user-images.githubusercontent.com/45444757/172593033-3c4705de-54f0-4279-8061-f11f8fd5d3bf.png" width='70%' height='70%'>

- 기능

  - Popular Videos를 메인 화면에서 확인할 수 있습니다.  : Youtube Most Popular API를 활용해 구현했습니다. 
  
  - 각 Video는 클릭 시 동영상 재생 + 동영상 정보확인이 가능합니다. : Youtube API를 활용해 구현했습니다. 
  
  - 검색창을 이용해 원하는 영상을 검색할 수 있습니다.  : Youtube Search API를 활용해 구현했습니다. 검색 값을 받아와 서버와 통신합니다.

📝 메모

  - Postman을 이용해 API활용을 쉽게 하는 것을 연습해보았습니다.

  - React 를 처음 사용해보았습니다. Hook 개념, 재사용성을 고려해 기능 별로 Component를 분리하는 것, API를 사용하며 보안과 확장성을 위해 Dependency Injection을 하는 것 등 새로 배운 개념이 많았습니다. 익숙해지며 사용할 때까지 많은 연습이 필요할 것 같습니다.
